## Project Introduction

### Overview

My **second** practical backend project. The tutorial comes from the **Heima uploader** on **Bilibili**. The backend was written line by line by myself, and although it's not perfect, it's worth commemorating!  
The frontend code comes from **Heima's public materials on their official account**. For the **convenience** of **joint debugging** between frontend and backend, **theoretically** I should have created another separate repository, but for **convenience**, I directly placed the **frontend code** in the **backend repository**.
